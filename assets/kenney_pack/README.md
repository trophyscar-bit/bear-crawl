# Kenney asset packs (CC0 — free, public domain)

Downloaded directly from kenney.nl via curl. All CC0 (use freely, no attribution required).

- **tiny-dungeon/**   — 16x16 roguelike dungeon tiles (132 tiles) + Tilemap sheet
- **tiny-town/**      — 16x16 top-down town/exterior tiles (132) + Tilemap
- **top-down-shooter/** — top-down characters + weapons + tiles (587 PNGs)
- **particle-pack/**  — VFX particle sprites (smoke, fire, magic, etc.) transparent + black-bg
- **pixel-shmup/**    — shmup ships (24) + tiles (120) + projectiles

Slice big sheets with: `python tools/slice_sheet.py <sheet> --out <dir> --grid 16 16`
